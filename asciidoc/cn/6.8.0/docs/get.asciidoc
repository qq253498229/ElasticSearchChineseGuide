=== Get API

Get接口可以根据id从索引中获取JSON文档。
下面就是一个获取JSON格式文档的例子，它的条件是index为twitter，type是 `_doc` ，id是1:

[source,java]
--------------------------------------------------
GetResponse response = client.prepareGet("twitter", "_doc", "1").get();
--------------------------------------------------

有关Get操作的更多信息，请查看 REST {ref}/docs-get.html[get] 。
